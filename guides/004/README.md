## Manage files in the Staging Area
- spesifik ke suatu file
```js
git add <file>
```
- seluruh list file 
'''js
git add . 
'''
- mengeluarkan file dari staging area
'''js
git rm --cached <file>
'''
- mengeluarkan directory dari staging area
'''js
 git rm -r --cached <directory>
'''
